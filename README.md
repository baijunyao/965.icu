# 965.happy

“965” 工作制，即每天早 9 点到岗，一直工作到晚上 6 点。每周工作 40 天。

“965”工作制的周工作时间为最低 40 小时。


## 注意事项

 - **965 并不意味着不加班，但是加班一定有调休、高额补贴**
